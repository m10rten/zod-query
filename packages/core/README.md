# Zod query

A query function tool to query data from zod schema's.

## Installation

### NPM

```bash
npm install zod-query
```

### Yarn

```bash
yarn add zod-query
```

### PNPM

```bash
pnpm add zod-query
```

## Documentation

For more information, check out the [documentation](https://github.com/m10rten/zod-query)
